# @opendestiny/astro-wnl

天文历法计算核心库 (Astronomical and traditional Chinese calendar computation core)

## 功能

- 农历/公历转换
- 节气计算
- 天文星历计算
- 传统历法工具

## 安装

```bash
pnpm add @opendestiny/astro-wnl
```

## 使用

```typescript
import { /* 相关函数 */ } from '@opendestiny/astro-wnl';
```

## 开发

```bash
pnpm install
pnpm dev
```
